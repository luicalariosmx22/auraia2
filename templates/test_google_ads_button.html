Prueba de integración del botón de actualización de datos

Métricas de Google Ads

Las métricas se actualizan automáticamente cada semana, pero puedes obtener los datos de los últimos 7 días manualmente usando el botón a continuación.

Esta es una página de prueba. En producción, el botón debe integrarse en el panel de Google Ads existente.

Instrucciones de integración

  1. Añade el componente JavaScript a tu página de panel de Google Ads
  2. Crea un contenedor con ID "google-ads-update-container" donde quieras mostrar el botón
  3. Inicializa el componente como se muestra en este ejemplo
<!-- Añade este código al final de tu página -->
<script src="/static/js/components/google_ads_update_button.js"></script>
<script>
    document.addEventListener('DOMContentLoaded', function() {
        // Inicializar el botón de actualización
        const updateButton = new GoogleAdsUpdateButton('google-ads-update-container');
    });
</script>
